Updating - Windows

UPDATING GIT
Git may already be installed on your system for several reasons: your operating system might come with Git pre-installed, a software tool you installed could include Git as a dependency, or you might have installed Git previously and forgotten about it. Regardless of the reason, if you want to update Git for your Windows machine, then you can use the following commands:

CMD - PowerShell [ If existing Git version is between 2.14.2 and 2.16.1 ]

git update

CMD - PowerShell [ If existing Git version is greater than 2.16.1 ]

git update-git-for-windows

Updating - MacOS

UPDATING GIT
Git may already be installed on your system for several reasons: your operating system might come with Git pre-installed, a software tool you installed could include Git as a dependency, or you might have installed Git previously and forgotten about it. Regardless of the reason, if you want to update Git for your MacOS machine, then you first need to make sure that the Homebrew package manager is up to date by using the following command:

Zsh - Bash

brew update

Once the process is finished, you can update git using the following command:

Zsh - Bash

brew upgrade git

Updating - Linux

UPDATING GIT
Git may already be installed on your system for several reasons: your operating system might come with Git pre-installed, a software tool you installed could include Git as a dependency, or you might have installed Git previously and forgotten about it. Regardless of the reason, if you want to update Git for your Linux machine, then use the appropriate commands from the list below, depending on your distribution and package manager (Please remember to update the package manager first before attempting to update Git itself):

DEBIAN/UBUNTU
For Debian-based distributions, which use the apt (Advanced Package Tool) package manager, use the following command to update the package manager and Git simultaneously:

sudo apt update && sudo apt upgrade -y git

Fedora
For Fedora-based distributions, which use the dnf (Dandified YUM) package manager, use the following command to update the package manager and Git simultaneously:

sudo dnf upgrade --refresh git

Arch
For Arch-based distributions, which use the pacman package manager, use the following command to update the package manager and Git simultaneously:

sudo pacman -Syu git
