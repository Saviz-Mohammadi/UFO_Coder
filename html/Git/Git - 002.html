Installation - Windows

CHECKING FOR INSTALLATION
Git may already be installed on your system for several reasons: your operating system might come with Git pre-installed, a software tool you installed could include Git as a dependency, or you might have installed Git previously and forgotten about it. Regardless of the reason, if you want to check whether Git is installed—whether to verify an existing installation or confirm a new one—you can use the following command on Windows:

CMD - PowerShell

git --version

INSTALLING GIT
The easiest way to install Git on a Windows machine is to download the official installer from the Download for Windows page. Once downloaded, double-click the installer and follow the installation wizard to customize Git settings according to your preferences. It is generally recommended to stick with the default settings. After completing the installation, you should be able to use Git from the command line. To confirm that Git is properly installed, refer back to the section of CHECKING FOR INSTALLATION.

Installation - MacOS

CHECKING FOR INSTALLATION
Git may already be installed on your system for several reasons: your operating system might come with Git pre-installed, a software tool you installed could include Git as a dependency, or you might have installed Git previously and forgotten about it. Regardless of the reason, if you want to check whether Git is installed—whether to verify an existing installation or confirm a new one—you can use the following command on MacOS:

Zsh - Bash

git --version

INSTALLING GIT
The easiest way to install Git on a MacOS machine is by using the Homebrew package manager. First, ensure that Homebrew is installed. You can find the installation command at the top of the Homebrew homepage; here it is for your convenience:

Zsh - Bash

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

After installing Homebrew, search for a package named git. You should find the corresponding command to download the Git package. I also provide the command here for your convenience:

Zsh - Bash

brew install git

If you prefer not to use the Homebrew package manager, you can visit the Download for macOS page for alternative methods to install Git on MacOS. Once the installation is complete, you can confirm it by referring to the CHECKING FOR INSTALLATION section.

Installation - Linux

CHECKING FOR INSTALLATION
Git may already be installed on your system for several reasons: your operating system might come with Git pre-installed, a software tool you installed could include Git as a dependency, or you might have installed Git previously and forgotten about it. Regardless of the reason, if you want to check whether Git is installed—whether to verify an existing installation or confirm a new one—you can use the following command on Linux:

Bash

git --version

INSTALLING GIT
The best way to install Git on a Linux machine is by using the native package manager specific to your distribution. Given the diversity of Linux distributions, each with its own package manager and installation command, it's impractical to list every command here. Fortunately, the Download for Linux and Unix page provides a comprehensive list of popular distributions along with their respective package managers and installation commands. Once the installation is complete, you can confirm it by referring to the CHECKING FOR INSTALLATION section.
